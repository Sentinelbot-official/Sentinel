Frequently Asked Questions

üí∞ Pricing & Free Features

Is Sentinel really 100% free?

Yes! Every single feature is completely free. No premium tiers, no hidden costs, no paywalls. We're committed to keeping Sentinel free forever because Discord security shouldn't be behind a paywall.

How do you make money if it's free?

Sentinel is open-source and community-driven. We accept donations but don't require them. The goal is to provide better security for everyone, not to profit. If the project grows, we may explore optional enterprise support or partnerships, but core features will always be free.

What's the catch?

No catch! Sentinel is newer than Wick, so we don't have the massive user base yet. But the code is open source - you can verify everything yourself. We're building trust through transparency, not marketing budgets.

üÜö Sentinel vs Wick

How is Sentinel better than Wick?

Sentinel exceeds Wick in several key areas:

  • 4 anti-raid algorithms vs Wick's 1
  • AI-powered features - Wick has none
  • Predictive threat detection - Stops attacks before they happen
  • Automatic hourly backups - Wick requires manual recovery
  • 10x faster recovery - Parallel processing
  • 100% free - Wick charges for premium
  • Open source - Verifiable security

Can I use both Sentinel and Wick?

Absolutely! Many users run both side-by-side to compare. They work together without conflicts. We're confident that after testing both, you'll prefer Sentinel.

Will Sentinel work if I already have Wick?

Yes! They can run simultaneously. Sentinel won't interfere with Wick's operations. This lets you test Sentinel risk-free before making a switch.

üîí Security & Privacy

Is Sentinel safe? Can I trust it?

Sentinel is 100% open source on GitHub. You can read every line of code and verify exactly what it does. Unlike closed-source bots, there's nothing hidden. Security researchers can audit the code, and you can self-host if you want complete control.

What data does Sentinel collect?

Sentinel only collects data necessary for features to work:

  • Server configuration (your settings)
  • Moderation logs (cases, warnings)
  • Security events (threats detected)
  • User behavior (for AI analysis, if enabled)

We don't sell data, share with third parties, or use it for advertising. Full privacy policy available on GitHub.

What happens to my data if I remove Sentinel?

All data is deleted within 30 days of bot removal. You can also request immediate deletion using /privacy export before removing the bot.

‚öôÔ∏è Setup & Usage

How long does setup take?

2 minutes! Invite the bot, run /setup preset, and you're done. Sentinel auto-configures based on your server type.

Do I need technical knowledge?

Nope! Sentinel is designed for everyone. The /setup preset command handles everything automatically. Advanced users can dive into detailed configuration, but it's not required.

What permissions does Sentinel need?

Administrator permission is recommended for full functionality. Minimum permissions: Manage Server, Manage Roles, Manage Channels, Ban Members, Kick Members, Manage Messages, Read Message History.

Can Sentinel handle large servers?

Yes! Sentinel uses automatic sharding and is cluster-ready. It scales from small communities to servers with 100,000+ members.

ü§ñ AI Features

How does the AI work?

Sentinel uses pattern recognition and behavioral analysis to:

  • Detect suspicious user behavior
  • Predict threats before they happen
  • Provide security recommendations
  • Prioritize moderation actions

The AI learns from patterns across all Sentinel servers (anonymized) to improve detection accuracy.

Does the AI make false positives?

Sentinel has a whitelist system to prevent false positives. Trusted users are exempt from AI analysis. The AI also uses confidence scoring - it only takes action at 60%+ confidence.
